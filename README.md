# FoodVision11

## Описание Задачи

В данном проекте решается задача **многоклассовой классификации изображений**, сравниваются шесть архитектур глубокого обучения:

1. **Нейронная сеть (EfficientNet-B0)**
2. **Нейронная сеть (EfficientNet-B1)**
3. **Нейронная сеть (EfficientNet-B2)**
4. **Нейронная сеть (EfficientNet-B3)** 
5. **Легкая нейросеть (MobileNetV2)** 
6. **Классическая нейросеть (ResNet-18)** 


##  Подготовка Данных

- **Источник:** Датасет Food11 с Kaggle (11,000 обучающих, 1,100 тестовых изображений)
- **Классы:** 11 категорий продуктов питания (apple_pie, cheesecake, chicken_curry и т.д.)
- **Распределение:** Сбалансированное — по 1,000 изображений на класс в обучении, по 100 в тесте

### Предобработка:

- **Аугментация данных:**
    - Случайное изменение размера и кроп (*RandomResizedCrop*, масштаб 0.7–1.0)
    - Горизонтальные и вертикальные отражения (*RandomHorizontalFlip*, *RandomVerticalFlip*)
    - Случайные повороты до 25°
    - Изменение яркости, контраста, насыщенности (*ColorJitter*)
    - Нормализация по ImageNet статистике

- **Разделение:**
    - Обучающая выборка: 11,000 изображений
    - Тестовая выборка: 1,100 изображений
    - Размер батча: 16–32 (зависит от модели)


##  Архитектуры Моделей

### EfficientNet (B0, B1, B2, B3)

- **Основа:** Масштабируемая архитектура EfficientNet
- **Выходной слой:** Линейный слой с 11 нейронами (классы)
- **Инициализация:** Предварительно обученные веса ImageNet

| Модель | Параметры | Размер | Input Size |
|--------|-----------|--------|-----------|
| B0 | 5.3M | 21 MB | 224x224 | 
| B1 | 7.8M | 31 MB | 240x240 |
| B2 | 9.2M | 37 MB | 260x260 | 
| B3 | 12.2M | 49 MB | 300x300 | 

### MobileNetV2

- **Основа:** Инвертированные остатки
- **Параметры:** 3.5M (самая легкая)
- **Размер:** 14 MB
- **Input Size:** 224x224
- **Назначение:** Мобильные приложения, edge-devices

### ResNet-18

- **Основа:** Классическая архитектура с остаточными блоками
- **Параметры:** 11.2M
- **Размер:** 45 MB
- **Input Size:** 224x224
- **Назначение:** Baseline для сравнения
